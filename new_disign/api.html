Методы пользователей

Получить количество пользователей

Возвращает количество пользователей, зарегистрированных в системе

Path: api/get_users_count
Input: none
Output: integer

Методы тестов

Получить количество тестов

Возвращает количество открытых тестов в системе

Path: api/get_tests_count
Input: none
Output: integer

Получить список тестовв

Возвращает список всех открытых тестов формате JSON

Path: api/get_tests_list
Input: none
Output: json

Получить количество тестов в группе

Возвращает количество открытых тестов в одной группе

Path: api/get_tests_count_by_group/int:id_group
Input: integer
Output: integer / null

Получить список тестов в группе

Возвращает список открытых тестов в одной группе в формате JSON

Path: api/get_tests_list_by_group/int:id_group
Input: integer
Output: json / null

Методы прохождений

Получить количество прохождений

Возвращает количество прохождений всех тестов в системе

Path: api/get_results_count
Input: none
Output: integer

Получить список прохождений

Возвращает список прохождений всех тестов в системе в формате JSON

Path: api/get_results_list
Input: none
Output: json

Получить количество прохождений одного теста

Возвращает количество прохождений одного определённого теста в системе

Path: api/get_tests_count_by_test/int:id_test
Input: integer
Output: integer / null
